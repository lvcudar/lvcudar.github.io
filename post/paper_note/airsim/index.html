Read Paper 《AirSim: High-Fidelity Visual and Physical Simulation for Autonomous Vehicles》

2019-11-02
Paper Note CV

Contents

    • TL;DR
    • Dataset
    • Architecture
    • Algorithm
    • Thoughts

URL: https://microsoft.github.io/AirSim/

TL;DR

FSR2017 一个自动驾驶模拟器，为四旋翼飞行器通过类似车的方式提供模拟环境。 有虚幻和Unity的版本，但是需要注册才可以获得。

Dataset

这项工作主要是为四旋翼飞行棋提供逼真的物理参数和操控反馈，并没有携带复杂的预置的动态交通环境。

Architecture

模拟器由环境模块、车辆(智能体)模块、物理引擎、传感器模块、渲染接口、公共API层和机械固件接口层组成。

Algorithm

四旋翼飞行器按照刚体力学来计算运动状态。 重力、坐标等都有精确的模拟，帧率也很高。

Thoughts

这更像是一个电机传动的精确物理模拟器，而不是一个交通仿真环境。
